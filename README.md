# Learning Management System (LMS) using MongoDB

## Project Description
This project is a simple Learning Management System (LMS) developed as part of the 
Naan Mudhalvan Skill Development Program. It uses MongoDB to perform CRUD operations 
for managing course details.

## Technologies Used
- MongoDB
- Node.js
- Express.js
- React.js
- GitHub

## Features
- Add new courses
- View available courses
- Update course details
- Delete courses

## Database Details
Database Name: lmsDB  
Collection Name: courses  

## CRUD Operations
- Create: Insert course data into MongoDB
- Read: Retrieve course details
- Update: Modify course information
- Delete: Remove course records

## How to Run the Project
1. Install Node.js and MongoDB
2. Run MongoDB server
3. Install dependencies using npm install
4. Run backend using node server.js
5. Run frontend using npm start
